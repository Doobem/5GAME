# Five Fates

A roguelike adventure game where you battle through five challenging biomes, each with unique enemies and environments. Collect random powers and test your skills in this action-packed browser game.

## How to Play

Open `index.html` in any modern web browser to start playing.

### Controls

| Key | Action |
|-----|--------|
| WASD / Arrow Keys | Move |
| Space | Jump |
| Shift | Dash |
| 1-5 | Use Powers |

### Gameplay

- **Objective**: Defeat 15 enemies in each biome to advance
- **Progression**: Complete all 5 biomes to win
- **Powers**: Each run grants 5 random powers (attack, mobility, or defense)
- **Resources**: Manage your Health and Mana wisely

### Biomes

1. Ashen Wastes
2. Crystal Forest
3. Sunken Ruins
4. Iron Citadel
5. Celestial Rift

### Powers

- **Fireball** - Ranged fire attack
- **Ice Shard** - Quick ice projectile
- **Lightning** - High damage electric attack
- **Dash** - Quick movement burst
- **Shield** - Temporary invulnerability

## Features

- Procedurally selected powers per run
- Multiple enemy types (Sprite, Beast, Golem)
- Progressive difficulty across biomes
- Score tracking
- Sound effects
- Particle effects

## Technical Details

- Pure HTML5, CSS3, and JavaScript
- Canvas-based 2D rendering
- Web Audio API for sound effects
- No external dependencies required
